# vue-ssr

> vue.js server render simple

## Build Setup

``` bash
# install dependencies
``` 
npm install or yarn 
```

# build for production with minification
``` 
npm run build 
```

# start server 
``` 
node server.js
```

you can see result ``` http://localhost:8080 ```
